更新日志

版本演进记录以 GitHub Releases 为准，此处用于官网快速查看主要变化。

v0.1.3

2026-03-30
  • 修复 Windows 安装器“应用运行中提示”中文乱码问题（NSIS 按 UTF-8 读取脚本）。
  • 版本号升级为 0.1.3，并同步安装包命名与官网版本展示。

v0.1.2

2026-03-30
  • Windows 安装包命名升级为 `MassiveEdit-版本号-Setup.exe`。
  • NSIS 安装/卸载新增“程序运行中检测”，提示关闭后重试。
  • 官网静态页面上线（首页、下载、更新日志、隐私政策）。

v0.1.1

2026-03-30
  • 修复 Windows 中文文件名路径无法打开的问题（UTF-16 路径链路）。
  • 新增拖拽文件到主窗口直接打开。
  • Windows 可执行文件与安装器图标完善。
  • 自定义 NSIS 脚本接管打包流程，开始菜单改为单项直达启动。

v0.1.0

2026-03-29
  • 首个跨平台可用版本，支持大文件浏览与基础编辑。
  • 提供查找替换、撤销重做、编码与换行格式管理。
  • 提供 Windows/macOS/Linux 构建与打包脚本。
