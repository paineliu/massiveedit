Large File Editor

面向超大文本文件的编辑体验

MassiveEdit 是一款基于 Qt 开发的大文件文本编辑器，强调启动速度、滚动流畅性与稳定编辑能力。
由五道题（北京）科技有限公司持续维护与迭代。

MIT 开源协议
Win / macOS / Linux 平台支持

核心能力

围绕“大文件可读、可搜、可改”设计，优先保障稳定性与响应速度。

大文件高效加载

分块读取与缓存策略，减少整体加载等待，提升打开与滚动体验。

渐进索引与搜索

支持异步检索与结果定位，面向大体量文本保持可操作性。

可靠编辑基础

提供撤销/重做、编码与换行格式管理，为后续能力扩展留足空间。
